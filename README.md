# BreachKit

## Overview

BreachKit is a web application that allows you to perform reconnaissance and vulnerability detection on a target organization.

The goal is to provide a simple, easy to use, and scalable platform for performing reconnaissance and vulnerability detection and reporting on a target organization.


### Tech Stack

#### Backend
- **Litestar**: A fast ASGI web framework with optional support for Starlette middleware and the use of Pydantic ORM models for routing.
- **SurrealDB**: A scalable, distributed, general-purpose database that is easy to use.
- **Ollama**: An open-source, self-hosted inference API for running LLMs.
- **SeaweedFS**: A simple and highly scalable distributed file system.
- **Docker**: To containerize the application and its dependencies.
- **Bulma**: A modern CSS framework based on Flexbox.



#### Recon Tools

- **BBot**: A multi-modal scanner for performing reconnaissance and vulnerability detection.
- **Nuclei**: A fast, open-source, and next-generation framework for discovering vulnerabilities.
- **Nmap**: A powerful network scanning tool.
